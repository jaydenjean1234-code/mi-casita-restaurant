## 🔧 Setup & Deployment

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/jaydenjean1234-code/mi-casita-restaurant.git
   ```

2. Open `index.html` in your web browser

3. No build tools or dependencies required!

### Deployment Options

#### GitHub Pages (Free)
1. Push to your GitHub repository
2. Go to Settings → Pages
3. Select `main` branch as source
4. Your site will be live at `https://yourusername.github.io/mi-casita-restaurant/`

#### Netlify (Free)
1. Connect your GitHub repository
2. Set build command to empty (static site)
3. Deploy instantly
4. Get automatic deployments on each push

#### Vercel (Free)
1. Import your GitHub repository
2. Deploy with one click
3. Get automatic SSL and CDN
